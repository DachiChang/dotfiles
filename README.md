# dotfiles

## Install

1. clone dotfiles to ~/
2. install stow (brew install stow)
3. cd dotfiles
  - stow zsh
  - stow nvim
  - stow tmux
  - stow misc
  - stow alacritty
  - stow htop
  - stow opencode

## Uninstall

1. cd dotfiles
  - stow -D zsh
  - stow -D nvim
  - stow -D tmux
  - stow -D misc
  - stow -D alacritty
  - stow -D htop
  - stow -D opencode

## Tmux

KeyBind
  - F11 previous window
  - F12 next window

PluginManager
  - prefix + I, to install plugins
  - prefix + U, to update plugins
  - prefix + alt + u, to clean plugins

SessionManagement
  - prefix + ctrl + s, to save session environment
  - prefix + ctrl + r, to restore session environment
